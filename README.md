🌌 AstroHub
Explore a Universe of Games

AstroHub is a sleek, space‑themed gaming platform where players can launch into a growing galaxy of browser‑based games. Designed for speed, simplicity, and exploration, AstroHub brings every adventure together in one futuristic hub. More games and features will be added as the platform expands — prepare for liftoff.

🚀 Features
🌠 Space‑themed interface inspired by cosmic exploration

🕹️ Browser‑based games that run instantly with no downloads

🪐 Organized game library with room to grow

⚡ Fast, lightweight design for smooth gameplay

🌌 Expanding universe — new games and updates coming soon

🪐 Games
The /games directory will host all playable titles.
Games will be added soon as AstroHub expands its galaxy.

🌙 Theme & Vision
AstroHub is built around the idea of exploration — a digital cosmos where every game is a new world waiting to be discovered. The goal is to create a platform that feels alive, futuristic, and fun, with a clean interface that makes jumping between adventures effortless.
